Powershel Comandos Básicos

Conceder Acesso a Diretórios

Abra o PowerShell como administrador e execute os seguintes comandos para conceder acesso a diretórios:

Exemplo 1: Copiar um descritor de segurança de um arquivo para outro:

$DogACL = Get-Acl -Path "C:\Dog.txt"
Set-Acl -Path "C:\Cat.txt" -AclObject $DogACL

Exemplo 2: Copiar um descritor de segurança para múltiplos arquivos:

$Acl = Get-Acl -Path "C:\Users\nucleomap\OZ3_ServidorGeral\Automacao - Eplan\Parts\MSSQL16.EPLAN\MSSQL"
Get-ChildItem -Recurse -Force | Set-Acl -AclObject $Acl

Exibir Conteúdo de Arquivo

Use o comando abaixo para exibir o conteúdo de um arquivo:

type "Filename.txt"

Comando Echo

Este comando insere um texto em um arquivo. Se o arquivo não existir, ele será criado. Caso já exista, o texto será adicionado ao final do arquivo:

echo "Inserir texto" >> Filename.txt

Gerar Chave Pública

Para gerar uma chave SSH e adicioná-la ao GitHub, utilize os seguintes comandos:

ssh-keygen -t ed25519 -C "your_email@exemplo.com"

Ou, para gerar uma chave RSA:

ssh-keygen -t rsa -b 4096 -C "your_email@exemplo.com"
