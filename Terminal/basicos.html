Terminal Comandos Básicos

Listar diretórios

Use o comando abaixo para listar todos os arquivos e pastas em um diretório: 

dir

Mudar diretório

Use o comando abaixo para mudar para um diretório específico:

cd [nome-do-diretorio]

Copiar arquivos

Use o comando abaixo para copiar arquivos de um local para outro:

copy [arquivo-fonte] [destino]

Mover arquivos

Use o comando abaixo para mover arquivos de um local para outro:

move [arquivo-fonte] [destino]

Apagar arquivos

Use o comando abaixo para deletar arquivos:

del [nome-do-arquivo]

Criar diretório

Use o comando abaixo para criar um novo diretório:

mkdir [nome-do-diretorio]

Renomear arquivos

Use o comando abaixo para renomear arquivos:

ren [nome-antigo] [nome-novo]

Ver informações do sistema

Use o comando abaixo para exibir informações detalhadas sobre o sistema: 

systeminfo

Limpar tela

Use o comando abaixo para limpar a tela do terminal:

cls

Ver endereço IP

Use o comando abaixo para exibir informações sobre a rede, incluindo o endereço IP: 

ipconfig

Testar conexão

Use o comando abaixo para testar a conectividade com outro dispositivo na rede: 

ping [endereço]

Ver rota de pacotes

Use o comando abaixo para exibir a rota que os pacotes percorrem até um destino: 

tracert [endereço]

Ver tarefas em execução

Use o comando abaixo para exibir todas as tarefas em execução no sistema: 

tasklist

Finalizar tarefa

Use o comando abaixo para finalizar uma tarefa específica:

taskkill /IM [nome-do-processo] /F

Verificar e reparar disco

Use o comando abaixo para verificar e reparar erros no disco:

chkdsk

Formatar disco

Use o comando abaixo para formatar um disco:

format [unidade:]

Ver tabelas de roteamento

Use o comando abaixo para exibir a tabela de roteamento da rede:

route print

Ver compartilhamentos de rede

Use o comando abaixo para exibir todos os compartilhamentos de rede no sistema: 

net share

Sincronizar horário

Use o comando abaixo para sincronizar o relógio do sistema com um servidor de horário: 

w32tm /resync

Configurar firewall

Use o comando abaixo para configurar as regras do firewall do Windows: 

netsh advfirewall firewall

Ver espaço em disco

Use o comando abaixo para exibir informações sobre o espaço em disco: 

wmic logicaldisk get size,freespace,caption

Abrir editor de registro

Use o comando abaixo para abrir o Editor de Registro do Windows:

regedit

Abrir gerenciador de tarefas

Use o comando abaixo para abrir o Gerenciador de Tarefas:

taskmgr

Ver versão do Windows

Use o comando abaixo para exibir a versão do Windows:

ver

Desligar sistema

Use o comando abaixo para desligar o sistema:

shutdown /s

Reiniciar sistema

Use o comando abaixo para reiniciar o sistema:

shutdown /r

Ver permissões de arquivos

Use o comando abaixo para exibir as permissões de um arquivo:

icacls [nome-do-arquivo]

Alterar permissões de arquivos

Use o comando abaixo para alterar as permissões de um arquivo:

cacls [nome-do-arquivo] /P [usuário]:[permissões]

Listar conexões e portas

Use o comando abaixo para exibir conexões e portas:

netstat -a

Listar programas

Use o comando abaixo para exibir programas que estão usando as conexões: 

netstat -b

Listar conexões em forma numérica

Use o comando abaixo para exibir conexões em forma numérica:

netstat -n

Ver número de serie da BIOS

Use o comando abaixo para exibir o número de serie da BIOS:

wmic bios get serialnumber
