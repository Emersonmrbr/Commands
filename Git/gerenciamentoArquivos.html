Guia Essencial de Comandos Git

Gerenciamento de Arquivos e Commits

Adicionar arquivos ao índice

git add

Adiciona um arquivo ao índice de preparação para o commit. O Git começará a rastrear o arquivo e incluirá as alterações no próximo commit.

Adicionar todos os arquivos modificados

git add .

Adiciona todos os arquivos modificados e novos ao índice, preparando-os para o próximo commit.

Criar um commit com uma mensagem descritiva

git commit -m "Mensagem do commit"

Cria um commit com os arquivos adicionados ao índice e uma mensagem que descreve as alterações realizadas. Essa mensagem ajuda a documentar o que foi feito no commit.

Histórico de Commits

Visualizar histórico de commits

git log

Exibe o histórico de commits no repositório, incluindo informações como o identificador do commit (hash), o autor, a data e a mensagem do commit.

Filtrar histórico pelo autor

git log --author="Nome do Autor"

Filtra o histórico de commits para mostrar apenas os commits realizados por um autor específico.

Exibir os últimos N commits

git log -n N

Exibe os últimos N commits do repositório, útil para obter uma visão rápida das alterações mais recentes.

Exibir histórico de um arquivo específico

git log

Exibe o histórico de commits relacionados a um arquivo específico, útil para revisar alterações feitas em um arquivo específico ao longo do tempo.

Visualizar contagem de commits por autor

git shortlog -sn

Exibe uma contagem de commits agrupada por autor, útil para ver quem mais contribuiu para o projeto.

Exibir histórico em formato gráfico

git log --oneline --graph --decorate

Exibe o histórico de commits de forma compacta e com uma representação gráfica das branches, proporcionando uma visão clara da estrutura do repositório.

Exibir histórico em uma linha por commit

git log --oneline

Exibe o histórico de commits de forma compacta, com um resumo de cada commit em uma única linha.

Exibir histórico em uma linha por commit com representação gráfica

git log --oneline --graph --all

Exibe o histórico de commits de forma compacta, com um resumo de cada commit em uma única linha com uma visualização gráfica.

Comparando Mudanças

Mostrar diferenças entre arquivos ou commits

git diff

Exibe as diferenças entre as versões dos arquivos, mostrando as linhas adicionadas ou removidas. Pode ser usado para ver as mudanças entre o estado atual e o último commit.

Comparar a área de stage com o último commit

git diff --cached

Mostra as diferenças entre os arquivos na área de stage e o último commit. Esse comando é útil para revisar as alterações preparadas antes de criar um commit.

Comparar dois commits específicos

git diff

Compara duas versões específicas do repositório, mostrando as diferenças entre esses commits.

Comparar duas branches

git diff ..

Compara duas branches específicas, mostrando as diferenças entre elas.

Exibir apenas os nomes dos arquivos modificados

git diff --name-only

Exibe apenas os nomes dos arquivos que foram modificados, sem mostrar as diferenças entre as versões.
