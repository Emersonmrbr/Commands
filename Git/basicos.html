Comandos Básicos do Git

Comandos Básicos

Obter Versão do Git

git --version

Exibe a versão do Git instalada no sistema, útil para confirmar a instalação e verificar a versão atual.

Obter Ajuda sobre Comandos do Git

git --help

Lista os comandos disponíveis no Git e abre o manual de ajuda detalhando o uso de cada comando. Pode ser seguido por um comando específico, como git help commit, para exibir ajuda desse comando.

Listar Configurações Globais

git config --global --list

Exibe todas as configurações globais definidas no sistema, como nome de usuário e email, permitindo verificar a aplicação dessas configurações.

Configurações de Usuário

Define o nome e o email que aparecerão nos commits feitos neste sistema:

git config --global user.name "Seu Nome"
git config --global user.email "seuemail@exemplo.com"

Essas configurações são essenciais para rastrear a autoria dos commits, associando-as aos commits sempre que o Git for usado neste computador.

Definir VSCode como Editor Padrão

git config --global core.editor "code --wait"

Define o VSCode como editor padrão para editar mensagens de commit e outros conteúdos solicitados pelo Git. O argumento --wait instrui o Git a aguardar o fechamento do editor antes de prosseguir.
