Git Tags

Trabalhando com Tags

Criar uma Tag

git tag <nome-da-tag>

Cria uma tag em um commit específico. As tags são usadas para marcar pontos importantes no histórico, como versões de lançamento.

Exibir Tags

git tag

Exibe uma lista de todas as tags no repositório local.

Excluir uma Tag Local

git tag -d <nome-da-tag>

Exclui uma tag do repositório local. Para removê-la também do repositório remoto, é necessário excluí-la lá.

Enviar Tag para o Repositório Remoto

git push origin <nome-da-tag>

Envia uma tag específica para o repositório remoto.

Enviar Todas as Tags para o Repositório Remoto

git push --tags

Envia todas as tags locais para o repositório remoto de uma só vez.

Listar Tags no Repositório Remoto

git ls-remote --tags origin

Lista todas as tags disponíveis no repositório remoto.

Listar Tags no Repositório Remoto Sem Link de Commit

git ls-remote --tags origin | findstr /v /c:"^{}"

Lista todas as tags no repositório remoto, excluindo duplicações de links para commits.

Excluir Tag no Repositório Remoto

git push origin --delete <nome-da-tag>

Exclui uma tag específica do repositório remoto.
