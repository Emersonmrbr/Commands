Desfazer alterações

Desfazendo Alterações

Desfazer alterações não commitadas

git checkout --

Desfaz as alterações feitas em um arquivo desde o último commit. Isso restaura o arquivo ao estado em que estava no commit anterior.

Desfazer um commit recente

git reset --soft HEAD~1

Desfaz o último commit, mas mantém as alterações no seu diretório de trabalho. Esse comando é útil se você deseja reverter rapidamente um commit, mas manter o conteúdo para ajustá-lo antes de refazer o commit.

Desfazer um commit e suas alterações

git reset --hard HEAD~1

Desfaz o último commit e todas as alterações feitas desde o commit. Use com cautela, pois as alterações serão perdidas.

Reverter um commit específico

git revert

Criar um novo commit que desfaz as alterações de um commit específico. Esse comando não altera o histórico de commits, mas cria um novo commit de reversão.
