Navegação e Gerenciamento de Branches

Mudar para uma branch específica

git checkout

Muda para uma branch específica no repositório. Se a branch não existir, o Git irá avisar que ela não foi encontrada.

Criar e mudar para uma nova branch

git checkout -b

Cria uma nova branch e muda para ela ao mesmo tempo. Esse comando é útil para criar novas funcionalidades ou corrigir bugs em paralelo ao desenvolvimento principal.

Comando alternativo para mudar de branch

git switch

Alternativa ao comando git checkout para mudar de branch. O comando switch é considerado mais intuitivo e foi introduzido em versões mais recentes do Git.

Voltar para um estado anterior do repositório

git checkout

Reverte o repositório para um estado anterior especificado por um commit. O repositório volta para o estado exato de como estava nesse commit.

Reverter um arquivo para o estado do último commit 

git checkout --

Reverte um arquivo para o estado do último commit, descartando as alterações feitas no arquivo desde o último commit.
