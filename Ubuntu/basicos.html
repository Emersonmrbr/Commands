Comandos Básicos do Ubuntu

Principais Comandos do Ubuntu

Aqui estão alguns dos comandos mais utilizados no Ubuntu para administração e gerenciamento do sistema:

Atualizar o Sistema

Para atualizar todos os pacotes do sistema, use os seguintes comandos:

sudo apt update
sudo apt upgrade

Instalar um Pacote

Para instalar um pacote usando o APT:

sudo apt install

Remover um Pacote

Para remover um pacote instalado:

sudo apt remove

Verificar o Status do Sistema

Para visualizar informações sobre o status do sistema:

top
htop

Listar Arquivos e Diretórios

Para listar arquivos e diretórios em um diretório:

ls

Para listar todos os arquivos (incluindo ocultos):

ls -a

Navegar Entre Diretórios

Para mudar de diretório:

cd

Para voltar ao diretório anterior:

cd ..

Ver o Conteúdo de um Arquivo

Para visualizar o conteúdo de um arquivo de texto:

cat

Para visualizar o conteúdo com a possibilidade de rolar:

less

Criar um Novo Arquivo

Para criar um novo arquivo de texto em um diretório:

touch

Editar um Arquivo

Para editar um arquivo de texto usando o editor de texto nano:

nano

Exibir o IP do Sistema

Para ver o endereço IP da sua máquina:

ip addr show

Verificar Espaço em Disco

Para verificar o espaço em disco disponível e utilizado:

df -h

Alterar Permissões de Arquivo

Para alterar as permissões de um arquivo ou diretório:

chmod

Alterar Proprietário de Arquivo

Para alterar o proprietário de um arquivo ou diretório:

chown

Reiniciar o Sistema

Para reiniciar o sistema:

sudo reboot

Desligar o Sistema

Para desligar o sistema:

sudo shutdown -h now

Expandir Disco na Máquina Virtual (VM)

Para expandir o disco de uma máquina virtual, siga os passos abaixo:

Reinicie a Máquina Virtual (VM) e instale as ferramentas do convidado

No terminal da VM, execute o seguinte comando para instalar o pacote cloud-guest-utils, que contém utilitários necessários para gerenciar discos:

sudo apt install cloud-guest-utils

Alterar a configuração de localidade (se não for o inglês)

Se o idioma do sistema não for o inglês, altere a configuração de localidade para evitar problemas com locais não ingleses. Use o comando abaixo para definir a localidade para o padrão inglês (C):

LC_ALL=C

Expandir a partição para o espaço livre disponível

Execute o seguinte comando para expandir a partição. O número "5" refere-se à partição específica (no caso, a partição 5), e você pode ajustá-lo conforme necessário para o seu sistema:

sudo growpart /dev/sda 5

Nota: Certifique-se de incluir um espaço entre partition e id, conforme mostrado no comando acima.

Redimensionar o sistema de arquivos para utilizar o novo espaço

Após expandir a partição, você precisará redimensionar o sistema de arquivos para aproveitar o espaço adicional. Execute o seguinte comando:

sudo resize2fs /dev/sda5

Nota: Não insira um espaço entre a partição e o identificador (/dev/sda5), conforme demonstrado no comando acima.

Esse procedimento irá garantir que sua máquina virtual tenha espaço extra disponível, permitindo que o sistema de arquivos utilize a capacidade adicional do disco expandido.
